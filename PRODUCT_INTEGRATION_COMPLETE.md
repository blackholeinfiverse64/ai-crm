## 📊 Integration Flow

```
User Action
    ↓
Frontend Component (Products.jsx)
    ↓
API Service (productAPI.js)
    ↓
HTTP Request → Backend (api_app.py)
    ↓
Database Query (SQLAlchemy)
    ↓
Response → Frontend
    ↓
State Update → UI Re-render
    ↓
Real-time Polling (Every 5s)
```

## 🚀 How to Use

### 1. Start Backend
```bash
cd backend
python api_app.py
```

### 2. Start Frontend
```bash
cd frontend
npm run dev
```

### 3. Access Application
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:8000`
- API Docs: `http://localhost:8000/docs`

### 4. Navigate to Products
- Click "Product Catalog" in sidebar
- View products (auto-updates every 5s)
- Create/Edit/Delete products
- Upload images
- Search & filter
